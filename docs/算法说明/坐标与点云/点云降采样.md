# 坐标与点云 / 点云降采样

## 功能说明

对点云数据进行降采样，减少点数量。支持体素网格降采样和随机降采样两种方式。

## 输入与输出

- 输入：点云数据
- 输出：降采样后的点云数据

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入点云 | PointCloudData | 是 | 无 | 输入点云数据 |
| voxel_size | double | 是 | 无 | 体素网格边长（VoxelGrid 方式） |
| ratio | double | 是 | 无 | 采样比例，范围 (0, 1]（RandomDownsample 方式） |

## 算法内容

### 类：PcDownsample

#### VoxelGrid(PointCloudData, voxel_size) -> unique_ptr\<PointCloudData\>

体素网格降采样。将三维空间划分为等大小的体素网格，每个体素内保留一个代表点（质心），从而均匀减少点数量。

#### RandomDownsample(PointCloudData, ratio) -> unique_ptr\<PointCloudData\>

随机降采样。按指定比例随机采样点云中的点，ratio 为保留点的比例。

### VoxelGrid 算法原理

1. 根据点云的包围盒和 voxel_size 计算体素网格的维度。
2. 将每个点分配到对应的体素中（通过坐标除以 voxel_size 取整计算体素索引）。
3. 对每个非空体素，计算其中所有点的质心作为代表点。
4. 输出所有代表点构成的降采样点云。

### RandomDownsample 算法原理

1. 根据输入点云总点数 N 和 ratio 计算目标点数 M = N × ratio。
2. 生成 M 个不重复的随机索引。
3. 按随机索引从原始点云中选取点。
4. 输出选取的点构成的降采样点云。

### 处理细节

- VoxelGrid 降采样结果与空间分布均匀，保留点云的整体形态。
- RandomDownsample 降采样结果具有随机性，多次执行结果不同。
- 两种方式均保留点的所有属性（强度、分类等）。
- VoxelGrid 中质心点的属性取体素内所有点属性的均值。

## 处理流程

1. 验证输入点云非空。
2. 根据调用方法执行对应降采样：
   - **VoxelGrid**：
     a. 计算点云包围盒。
     b. 按 voxel_size 划分体素网格。
     c. 将每个点分配到对应体素。
     d. 计算每个体素的质心点。
     e. 汇总所有质心点。
   - **RandomDownsample**：
     a. 计算目标点数。
     b. 生成随机索引。
     c. 按索引选取点。
3. 返回降采样结果。

## 输出说明

- VoxelGrid 输出点数为非空体素数量
- RandomDownsample 输出点数为原始点数 × ratio（取整）
- 输出点云属性与输入一致

## 结果解释

- VoxelGrid 降采样后点云分布均匀，适合保留整体形态
- voxel_size 越大，降采样程度越高，点数越少
- RandomDownsample 降采样后点云分布随机，适合快速缩减数据量
- ratio 越小，保留的点越少

## 适用场景

- 大规模点云的快速可视化预览
- 点云配准前的粗配准数据准备
- 减少点云数据存储和传输开销
- 深度学习训练数据预处理

## 推荐输出文件名

- `xxx_voxel_downsampled.las`
- `xxx_random_downsampled.las`

## 注意事项

- voxel_size 必须大于 0，单位与点云坐标单位一致
- ratio 必须在 (0, 1] 范围内，为 1 时输出与输入相同
- VoxelGrid 降采样结果确定性强，同一输入和参数结果一致
- RandomDownsample 结果具有随机性，如需可复现结果需固定随机种子
- VoxelGrid 对局部密集区域降采样效果显著，对稀疏区域影响较小
