# 栅格处理 / 掩膜转矢量

## 功能说明

将二值掩膜栅格转换为矢量多边形，使用 8 方向 Moore 邻域追踪算法提取边界，支持内环（孔洞）检测。

## 输入与输出

- 输入：二值掩膜（uint8 数组或单波段栅格）
- 输出：面类型矢量数据

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| mask | vector\<uint8\> | 是 | 无 | 二值掩膜数组 |
| width | int | 是 | 无 | 掩膜宽度 |
| height | int | 是 | 无 | 掩膜高度 |
| geotransform | double[6] | 是 | 无 | 仿射变换参数 |
| target_class | uint8 | 否 | `1` | 目标类别值 |

### ExecuteFromRaster 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| raster | RasterData | 是 | 无 | 输入栅格（取第一波段） |
| target_class | uint8 | 否 | `1` | 目标类别值 |

## 算法内容

### 8 方向 Moore 邻域追踪

使用 8 方向邻域追踪算法从掩膜中提取边界环。8 个方向定义如下：

```
方向编码：
  7  0  1
  6  P  2
  5  4  3

dx8 = [1, 1, 0, -1, -1, -1, 0, 1]
dy8 = [0, 1, 1, 1, 0, -1, -1, -1]
```

#### 外环追踪

1. 从左到右、从上到下扫描，找到左侧为非目标类的目标像素作为起始点。
2. 从方向 0 开始搜索，回溯到上一个进入方向的对向（`(dir + 6) % 8`）。
3. 顺时针搜索 8 个方向，找到第一个目标类邻居。
4. 移动到该邻居，更新方向，继续追踪。
5. 回到起始点时完成一个外环。

#### 内环追踪

1. 扫描位于外环内部、左侧为非目标类、右侧为目标类的像素。
2. 从方向 4（向左）开始追踪。
3. 仅保留有向面积为负的环（顺时针方向，表示内环）。

### 外环/内环判定

通过有向面积（Shoelace 公式）判定环的方向：

```
area = Σ(x_i * y_{i+1} - x_{i+1} * y_i) / 2
```

- area > 0：逆时针，外环
- area < 0：顺时针，内环

### 内环归属

使用 Winding Number 算法判断内环属于哪个外环：

1. 取内环的第一个点。
2. 对每个外环计算 Winding Number。
3. 选择嵌套层数最深（被最多外环包含）的外环作为归属。

### 坐标转换

像素坐标通过 GeoTransform 转换为地理坐标：

```
gx = gt[0] + px * gt[1] + py * gt[2]
gy = gt[3] + px * gt[4] + py * gt[5]
```

## 处理流程

1. 验证掩膜尺寸匹配。
2. 提取所有外环（8 方向追踪）。
3. 提取所有内环（孔洞）。
4. 关联内环到所属外环。
5. 构建矢量要素（外环 + 内环）。
6. 像素坐标转地理坐标。
7. 返回矢量数据。

## 输出说明

- 输出为面类型矢量数据
- 每个多边形可能包含内环（孔洞）
- 坐标为地理坐标（通过 GeoTransform 转换）

## 结果解释

- 每个多边形代表掩膜中一个连通的目标区域
- 内环表示目标区域内部的孔洞
- 顶点少于 4 个的环被过滤

## 适用场景

- 将 AI 分割掩膜转为矢量用于 GIS 分析
- 从二值分类图中提取地物边界
- 生成用于制图的面状要素

## 推荐输出文件名

- `xxx_polygon.gpkg`

### C++ 接口

```cpp
namespace gis_ai {

struct GIS_AI_API ContourPoint {
    double x;
    double y;
};

class GIS_AI_API MaskToPolygon {
public:
    MaskToPolygon() = default;

    // 从 uint8 掩膜数组转矢量
    std::unique_ptr<VectorData> Execute(const std::vector<uint8_t>& mask, int width, int height,
                                         const double* geotransform, uint8_t target_class = 1);

    // 从栅格数据转矢量（取第一波段）
    std::unique_ptr<VectorData> ExecuteFromRaster(const RasterData& raster, uint8_t target_class = 1);
};

} // namespace gis_ai
```

### 使用示例

```cpp
#include "ai/mask_to_polygon.h"
#include "io/raster_io.h"
#include "io/vector_io.h"

// 方式一：从栅格数据直接转矢量
auto raster = gis_ai::ReadRaster("mask.tif");
gis_ai::MaskToPolygon converter;
auto polygons = converter.ExecuteFromRaster(*raster);
gis_ai::WriteVector(*polygons, "output_polygon.gpkg");

// 方式二：从 uint8 掩膜数组转矢量
std::vector<uint8_t> mask = {/* ... */};
double geotransform[6] = {origin_x, pixel_w, 0, origin_y, 0, -pixel_h};
auto polygons2 = converter.Execute(mask, width, height, geotransform, 1);
gis_ai::WriteVector(*polygons2, "output_polygon.gpkg");
```

## 注意事项

- 掩膜尺寸必须等于 width × height
- target_class 指定要提取的类别值，默认为 1
- 追踪设有最大步数限制（width × height × 2），防止无限循环
- 内环检测依赖 Winding Number 算法，复杂嵌套场景可能不准确
- 输出多边形顶点为像素中心坐标转换后的地理坐标
