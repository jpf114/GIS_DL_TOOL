# 栅格处理 / 栅格镶嵌

## 功能说明

将多幅栅格影像拼接为一幅输出栅格，支持 5 种重叠区域处理策略。

## 输入与输出

- 输入：多个栅格数据（波段数必须一致）
- 输出：拼接后的栅格，建议 `.tif`

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入栅格列表 | vector\<RasterData\> | 是 | 无 | 至少 1 个输入栅格 |
| strategy | 枚举 | 否 | `Overwrite` | 镶嵌策略：Overwrite / First / Mean / Max / Min |
| nodata_value | float | 否 | `NaN` | NoData 值 |

## 算法内容

### 全局范围计算

遍历所有输入栅格的 GeoTransform，计算全局最小 X、最大 Y、最大 X、最小 Y，确定输出栅格的空间范围。以第一个栅格的像素尺寸为参考，计算输出栅格的宽高。

### 像素映射

每个输入栅格根据其 GeoTransform 计算在输出栅格中的像素偏移：

```
off_x = round((gt[0] - global_min_x) / ref_pixel_size_x)
off_y = round((global_max_y - gt[3]) / ref_pixel_size_y)
```

### 镶嵌策略

| 策略 | 说明 | 初始值 |
|------|------|--------|
| Overwrite | 后写入的值覆盖先写入的值 | NaN |
| First | 仅在当前位置为空时写入（保留第一个非空值） | NaN |
| Mean | 计算所有非空值的均值 | 0（累加器） |
| Max | 取所有非空值的最大值 | -∞ |
| Min | 取所有非空值的最小值 | +∞ |

### NaN 处理

- 输入中的 NaN 值不参与镶嵌计算
- Mean 策略下，无有效值的像素输出为 NaN

## 处理流程

1. 验证输入栅格列表非空。
2. 单个输入时直接返回副本。
3. 检查所有输入波段数一致。
4. 计算全局空间范围和输出尺寸。
5. 初始化输出栅格和累加缓冲区。
6. 逐输入栅格映射到输出网格，按策略写入。
7. Mean 策略下计算最终均值。
8. 返回镶嵌结果。

## 输出说明

- 输出波段数与输入一致
- 输出空间参考采用第一个输入栅格的投影
- 输出 GeoTransform 的左上角为全局最小 X、最大 Y

## 结果解释

- Overwrite：重叠区域取最后一个输入的值
- First：重叠区域取第一个输入的值
- Mean：重叠区域取所有输入的均值
- Max：重叠区域取最大值
- Min：重叠区域取最小值

## 适用场景

- 多景遥感影像合并成单幅成果图
- 分幅栅格整理
- 时间序列影像的合成（如取最大值合成）

## 推荐输出文件名

- `xxx_mosaic.tif`

### C++ 接口

```cpp
namespace gis_ai {

enum class GIS_AI_API MosaicStrategy {
    Overwrite,
    First,
    Mean,
    Max,
    Min
};

struct GIS_AI_API MosaicConfig {
    MosaicStrategy strategy = MosaicStrategy::Overwrite;
    float nodata_value = std::numeric_limits<float>::quiet_NaN();
};

class GIS_AI_API RasterMosaic {
public:
    RasterMosaic() = default;

    // 将多个栅格拼接为一幅输出栅格
    std::unique_ptr<RasterData> Execute(const std::vector<std::reference_wrapper<const RasterData>>& rasters,
                                         const MosaicConfig& config = MosaicConfig());
};

} // namespace gis_ai
```

### 使用示例

```cpp
#include "gis/raster_mosaic.h"
#include "io/raster_io.h"

auto r1 = gis_ai::ReadRaster("tile_1.tif");
auto r2 = gis_ai::ReadRaster("tile_2.tif");
auto r3 = gis_ai::ReadRaster("tile_3.tif");

// 默认 Overwrite 策略镶嵌
gis_ai::RasterMosaic mosaic;
auto result = mosaic.Execute({*r1, *r2, *r3});

// 使用均值策略镶嵌
gis_ai::MosaicConfig config;
config.strategy = gis_ai::MosaicStrategy::Mean;
config.nodata_value = 0.0f;
auto result2 = mosaic.Execute({*r1, *r2, *r3}, config);

gis_ai::WriteRaster(*result, "output_mosaic.tif");
```

## 注意事项

- 所有输入栅格的波段数必须一致
- 所有输入栅格应使用相同坐标系和相近的分辨率
- 像素尺寸为零或接近零时会报错
- Mean 策略需要额外的累加缓冲区，内存占用较高
