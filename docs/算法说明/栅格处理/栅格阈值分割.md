# 栅格处理 / 栅格阈值分割

## 功能说明

按阈值对栅格数据进行二值化分割，将像元值分为满足条件（1）和不满足条件（0）两类。支持单阈值和范围阈值两种模式。

## 输入与输出

- 输入：栅格数据（GeoTIFF）
- 输出：二值化栅格（1.0 / 0.0），建议 `.tif`

## 参数

### Execute（单阈值）

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入栅格 | RasterData | 是 | 无 | 输入栅格数据 |
| threshold | double | 是 | 无 | 分割阈值 |
| band_index | int | 否 | `0` | 波段索引（0 开始） |

### ExecuteRange（范围阈值）

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 输入栅格 | RasterData | 是 | 无 | 输入栅格数据 |
| min_val | double | 是 | 无 | 范围最小值 |
| max_val | double | 是 | 无 | 范围最大值 |
| band_index | int | 否 | `0` | 波段索引（0 开始） |

## 算法内容

### 单阈值分割

对指定波段的每个像元：

```
output = (value >= threshold) ? 1.0 : 0.0
```

NaN 值输出为 0.0。

### 范围阈值分割

对指定波段的每个像元：

```
output = (value >= min_val && value <= max_val) ? 1.0 : 0.0
```

NaN 值输出为 0.0。

### 输出特征

- 输出为单波段栅格
- 输出像元值仅为 1.0（满足条件）或 0.0（不满足条件）
- 输出尺寸和空间参考与输入一致

## 处理流程

1. 验证波段索引在有效范围内。
2. 创建输出栅格（单波段）。
3. 逐像元应用阈值判断。
4. 返回二值化结果。

## 输出说明

- 单波段浮点栅格
- 像元值 1.0 表示满足阈值条件
- 像元值 0.0 表示不满足阈值条件或为 NaN

## 结果解释

- 值为 1.0 的区域为分割出的目标区域
- 值为 0.0 的区域为背景或无效区域
- 可将结果作为掩膜用于后续掩膜转矢量操作

## 适用场景

- 从 NDVI 等指数影像中提取植被区域
- 从概率图或置信度图中提取高置信区域
- 从 DEM 中提取特定高程范围
- 将模型输出的概率掩膜二值化

## 推荐输出文件名

- `xxx_threshold.tif`

### C++ 接口

```cpp
namespace gis_ai {

class GIS_AI_API RasterThreshold {
public:
    RasterThreshold() = default;

    // 单阈值分割：value >= threshold 输出 1.0，否则 0.0
    std::unique_ptr<RasterData> Execute(const RasterData& input, double threshold, int band_index = 0);

    // 范围阈值分割：min_val <= value <= max_val 输出 1.0，否则 0.0
    std::unique_ptr<RasterData> ExecuteRange(const RasterData& input, double min_val, double max_val, int band_index = 0);
};

} // namespace gis_ai
```

### 使用示例

```cpp
#include "gis/raster_threshold.h"
#include "io/raster_io.h"

auto raster = gis_ai::ReadRaster("ndvi.tif");

// 单阈值分割：提取 NDVI >= 0.3 的区域
gis_ai::RasterThreshold thresholder;
auto result = thresholder.Execute(*raster, 0.3);

// 范围阈值分割：提取高程在 500~1500 之间的区域
auto dem = gis_ai::ReadRaster("dem.tif");
auto result2 = thresholder.ExecuteRange(*dem, 500.0, 1500.0);

gis_ai::WriteRaster(*result, "output_threshold.tif");
```

## 注意事项

- 波段索引从 0 开始
- 范围阈值要求 min_val ≤ max_val
- NaN 值统一输出为 0.0（视为不满足条件）
- 输出为浮点类型（1.0 / 0.0），非 Byte 类型
