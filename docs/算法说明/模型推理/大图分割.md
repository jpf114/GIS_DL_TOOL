# 模型推理 / 大图分割

## 功能说明

对超出模型输入尺寸的大幅遥感影像进行滑窗推理，通过重叠区混合消除接缝，并支持将分割结果矢量化输出。

## 输入与输出

- 输入：栅格数据（GeoTIFF）
- 输出：分割掩膜栅格（`.tif`）+ 矢量多边形（`.shp` / `.geojson`）

## 参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| tile_size | int | 否 | `512` | 切片大小（像素） |
| stride | int | 否 | `384` | 滑窗步长（像素），stride < tile_size 时产生重叠 |
| target_channels | int | 否 | `3` | 模型输入通道数 |
| normalize_mode | 枚举 | 否 | `ImageNet` | 归一化模式：None / ImageNet / MinMax01 / ZScore / Custom |
| input_is_uint8 | bool | 否 | `true` | 输入数据是否为 uint8 类型 |
| mean_r | float | 否 | `0.485` | ImageNet R 通道均值 |
| mean_g | float | 否 | `0.456` | ImageNet G 通道均值 |
| mean_b | float | 否 | `0.406` | ImageNet B 通道均值 |
| std_r | float | 否 | `0.229` | ImageNet R 通道标准差 |
| std_g | float | 否 | `0.224` | ImageNet G 通道标准差 |
| std_b | float | 否 | `0.225` | ImageNet B 通道标准差 |
| mask_threshold | float | 否 | `0.5` | 掩膜二值化阈值 |
| target_class | uint8 | 否 | `1` | 矢量化目标类别 |
| blend_mode | 枚举 | 否 | `Gaussian` | 混合模式：None / Linear / Gaussian |
| gaussian_sigma | float | 否 | `0.5` | Gaussian 混合的 sigma 系数 |
| skip_nodata | bool | 否 | `true` | 跳过全 NoData 的切片 |
| nodata_value | float | 否 | `0.0` | NoData 判定值 |
| min_polygon_area | double | 否 | `100.0` | 最小多边形面积过滤阈值 |
| simplify_tolerance | double | 否 | `1.0` | 矢量简化容差 |
| fix_topology | bool | 否 | `true` | 是否修复拓扑异常 |

## 算法内容

大图分割采用滑窗推理策略，将大幅影像按 `tile_size` 切分为多个小块，以 `stride` 为步长滑动。当 `stride < tile_size` 时，相邻切片之间存在重叠区域，通过混合权重函数消除接缝。

### 混合模式

- **None**：不混合，重叠区域直接覆盖。推理后对每个像素做 Sigmoid + Argmax 得到类别，直接写入结果。
- **Linear**：线性混合，重叠区域权重从 0 线性过渡到 1。权重公式：`w = (i + 1) / (overlap + 1)`，其中 `i` 为距边缘的像素距离。
- **Gaussian**：高斯混合，重叠区域权重按高斯函数过渡。权重公式：`w = 1 - exp(-d² / (2σ²))`，其中 `d` 为距边缘距离，`σ = gaussian_sigma × overlap`。

混合时对每个类别通道分别累加加权概率，最终按最大概率确定像素类别。

### 矢量化流程

1. 掩膜转矢量：8 方向 Moore 邻域追踪提取外环和内环
2. 过滤小多边形：按 `min_polygon_area` 移除面积过小的多边形
3. 简化：Douglas-Peucker 算法减少顶点
4. 拓扑修复：通过 GEOS Buffer(0) 修复非法几何
5. 属性添加：为每个多边形添加 class_id、feature_id、area、area_px、source 属性

## 处理流程

1. 加载模型并创建推理引擎。
2. 判断影像是否需要分块（宽或高 > tile_size）。
3. 若不需要分块，直接推理并输出。
4. 若需要分块：
   a. 计算所有切片位置和重叠信息。
   b. 预分配累加缓冲区和权重和缓冲区。
   c. 计算混合权重向量。
   d. 逐块处理：提取切片 → 预处理 → 推理 → 混合写入。
   e. 跳过全 NoData 切片（若启用）。
   f. 对累加结果做 Argmax 得到最终类别掩膜。
5. 将掩膜转为栅格输出。
6. 若需矢量化：掩膜转矢量 → 过滤 → 简化 → 拓扑修复 → 添加属性。

## 输出说明

- **栅格输出**：单波段 Byte 类型 GeoTIFF，像元值为类别编号。
- **矢量输出**：面类型 Shapefile 或 GeoJSON，每个多边形包含 class_id、feature_id、area、area_px、source 属性。

## 结果解释

- 栅格中每个像元的值表示该像素的预测类别编号（0 为背景）。
- 矢量中每个多边形代表一个连通区域，area 为地理面积，area_px 为像素面积。
- `SegmentationStats` 包含 total_tiles、skipped_tiles、inferred_tiles、total_inference_time_ms、polygon_count 等统计信息。

## 适用场景

- 大幅遥感影像的建筑物、道路、水体等地物提取
- 超出 GPU/CPU 内存限制的影像分割
- 需要矢量输出用于 GIS 分析的场景

## 推荐输出文件名

- `xxx_seg.tif`
- `xxx_seg.shp`

### C++ 接口

```cpp
namespace gis_ai {

enum class GIS_AI_API BlendMode {
    None,
    Linear,
    Gaussian
};

struct GIS_AI_API LargeImageSegConfig {
    int tile_size = 512;
    int stride = 384;
    int target_channels = 3;
    NormalizeMode normalize_mode = NormalizeMode::ImageNet;
    bool input_is_uint8 = true;
    float mean_r = 0.485f;
    float mean_g = 0.456f;
    float mean_b = 0.406f;
    float std_r = 0.229f;
    float std_g = 0.224f;
    float std_b = 0.225f;
    float mask_threshold = 0.5f;
    uint8_t target_class = 1;
    BlendMode blend_mode = BlendMode::Gaussian;
    float gaussian_sigma = 0.5f;
    bool skip_nodata = true;
    float nodata_value = 0.0f;
    double min_polygon_area = 100.0;
    double simplify_tolerance = 1.0;
    bool fix_topology = true;
};

struct GIS_AI_API SegmentationStats {
    int total_tiles = 0;
    int skipped_tiles = 0;
    int inferred_tiles = 0;
    double total_inference_time_ms = 0.0;
    double total_time_ms = 0.0;
    int polygon_count = 0;
    double total_polygon_area = 0.0;
    int class_counts[256] = {};
};

class GIS_AI_API LargeImageSeg {
public:
    explicit LargeImageSeg(const std::string& model_path);
    ~LargeImageSeg();

    LargeImageSeg(const LargeImageSeg&) = delete;
    LargeImageSeg& operator=(const LargeImageSeg&) = delete;

    // 大图分割，输出分割掩膜栅格
    std::unique_ptr<RasterData> Segment(const RasterData& input,
                                         const LargeImageSegConfig& config = LargeImageSegConfig());

    // 大图分割，输出矢量多边形
    std::unique_ptr<VectorData> SegmentToPolygon(const RasterData& input,
                                                   const LargeImageSegConfig& config = LargeImageSegConfig());

    // 大图分割，使用已有掩膜栅格进行矢量化
    std::unique_ptr<VectorData> SegmentToPolygon(const RasterData& input,
                                                   const RasterData& mask_raster,
                                                   const LargeImageSegConfig& config = LargeImageSegConfig());

    // 大图分割，直接输出到文件
    int SegmentToFile(const std::string& input_path,
                       const std::string& output_tif,
                       const std::string& output_shp = "",
                       const LargeImageSegConfig& config = LargeImageSegConfig());

    // 获取最近一次分割的统计信息
    SegmentationStats GetLastStats() const;

    // 设置进度回调函数
    void SetProgressCallback(std::function<void(int, int, const std::string&)> callback);
};

} // namespace gis_ai
```

### 使用示例

```cpp
#include "fusion/large_image_seg.h"
#include "io/raster_io.h"
#include "io/vector_io.h"

// 创建大图分割器
gis_ai::LargeImageSeg seg("model.onnx");

// 配置参数
gis_ai::LargeImageSegConfig config;
config.tile_size = 512;
config.stride = 384;
config.blend_mode = gis_ai::BlendMode::Gaussian;
config.mask_threshold = 0.5f;

// 设置进度回调
seg.SetProgressCallback([](int current, int total, const std::string& msg) {
    std::cout << "[" << current << "/" << total << "] " << msg << std::endl;
});

// 方式一：直接输出到文件（最简便）
seg.SegmentToFile("input.tif", "output_seg.tif", "output_seg.shp", config);

// 方式二：获取分割掩膜栅格
auto raster = gis_ai::ReadRaster("input.tif");
auto mask = seg.Segment(*raster, config);

// 方式三：获取矢量多边形
auto polygons = seg.SegmentToPolygon(*raster, config);
gis_ai::WriteVector(*polygons, "output_seg.gpkg");

// 查看统计信息
auto stats = seg.GetLastStats();
std::cout << "Total tiles: " << stats.total_tiles
          << ", Polygon count: " << stats.polygon_count << std::endl;
```

## 注意事项

- stride 必须 ≤ tile_size，否则自动调整为 tile_size（无重叠）
- Gaussian 混合效果最佳但计算量略高于 None
- 重叠区域越大（stride 越小），接缝越不明显，但推理次数越多
- 矢量化时 target_class 指定要提取的类别，默认为 1
- 拓扑修复使用 GEOS Buffer(0)，可能改变几何形状
